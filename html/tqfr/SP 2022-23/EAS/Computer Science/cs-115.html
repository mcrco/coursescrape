A very nice course on functional programming!
Similar to CS4
While you don't strictly need to have taken CS 4 (or otherwise learned OCaml) to take this class, this class can be a bit brisk, and Vanier doesn't really take stops to dig into common features like pattern matching or algebraic data types. Other than that, the class is interesting and the workload is fairly light. Vanier also explains monads quite clearly, which is good if you want to learn Haskell.
Great class. Take CS4 first. 6 labs with no midterm of final. Can rework each lab. Highly recommend.
you do not need to have taken cs 4 to do well in this course
This course is on Haskell, and I loved learning about a pure functional programming language. I would highly recommend this course for anyone who is looking to improve their coding style (and more people should, frankly) and for anyone who is interested in the functional programming paradigm in general. I don't think you need a textbook for this course, and I honestly forget if there even is one. Hoogle is your friend, but the course website and the lecture slides have the vast majority of the material you will need.
This is another great Vanier class. I did not take CS4, but enjoyed CS115 a lot. The class was very chill with no exams, reworks, and reasonable assignments. Would highly recommend as an advanced CS elective if you have an interest in functional programming.
A good course for CS option requirements. If you didn't like CS4 this probably is not the course for you. The first half of the course is pretty chill, and higher order functions were fun to learn. The course does get pretty difficult in the second half when you start learning about Monads so be prepared for that.
CS4 might be helpful, but is not a necessary prerequisite.
