The lectures are extremely explicit but boring. Prof. Bernardi has created very thorough notebooks which are extremely informative but his lecture style is fairly dry. The Friday courses are labs which function more as office hours for the homework assignments. Previous knowledge of Python is very helpful.
This is a minor detail but there was some confusion regarding the syllabus not matching the actual homework assignments. Also, it would have been helpful if the details of the midterm and final projects (dates, format, and so) were released sooner than they were.
Great introductory course in Python. I had almost no computational background but the class starts at a very simple level before building up and is conducive to a wide array of backgrounds and familiarities with coding.
This is a very useful course for an introduction to python that touches on more exciting topics later on; if you want to conduct data analysis and display your results in meaningful ways, this course will be helpful.
Great class, highly recommend this class to any graduate student. You end up having many different computational methods (and their examples) under your belt that can be used in your research no matter the field.
Great class, useful as an introduction to computational methods. Course load is very manageable, would recommend.
The course is a very quick overview of a lot of different topics. This was very useful to me, but it is not intended to delve into topics with a lot of depth.
Overall, a great course for an introduction to computational science (in Python). You don't need any prior knowledge of programming.

The work consists of 6 short homework assignments, a midterm group project (Jupyter notebook about a computational science topic + 20 min video presentation on the notebook), and a final group project (same as midterm but live presentation). Both projects don't involve too much work and are a great chance to explore a topic that you're interested in in more detail.
I went into this class being told it was going to be soo easy. It was not. It was hard, it took a long time, and I struggled. That said, I also had barely ever coded going in, and had forgot most linear algebra, so if you know those two, then you're all good.
The content is less "code" and more code implementations of math. Still useful, but a different skill set than you might be going for.
The lecture notes are very thorough and I benefited from spending the time to go through them by myself instead of going to lecture, as lecture is basically a skimming of the notes.
