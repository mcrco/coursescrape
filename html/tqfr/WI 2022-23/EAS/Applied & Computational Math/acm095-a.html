Well, you're probably taking this class because you have to, so make the best of it. Prof. Zuev's an *excellent* lecturer, and it's worth going to them, if you can handle the shorter lunch period (since class is MWF 11-12:15 for some unseemly reason). The lecture notes are also incredibly useful and well-written, so you can rely on them if you don't make it to as many lectures. (I never used the textbook.)

Overall, it's a solid enough introduction to complex analysis and differential equations, although overall I found the introduction by Ma 2 (practical) to be more intuitive. Much of it does just come down to pulling out the right equation, writing down a lot of things to show you're allowed to use it, using it, &c., &c. The grading was pretty harsh about that kind of accuracy this year. However, the head TA, Tea Freedman-Susskind, was actually one of the best TAs I've ever had, so that rather made up for it.

There's a set each week, as well as a midterm and a final (both closed book, you can only use your *own* notes). Like the sticker on the class says, it's 8 hours of homework per week. Some part of each set is in Matlab, but luckily they provide starter code, so it's not that bad even if you're not familiar with Matlab/coding in general.
This is probably the best-organized class I've taken so far; the materials are excellent and the content is well-taught. The math is also quite cool, although I don't find it particularly relevant to my major. One thing to note is that there is a Matlab problem on every set, and while it wasn't conceptually difficult, I ran into significant issues with Matlab-specific syntax that took hours to debug. This was extremely frustrating, so I'd recommend gaining some familiarity with Matlab (ACM 11 would be helpful) before taking this class.
Assignments take some time (appropriate for 12 unit course) but class resources like lecture videos, notes, and OH make it very doable if you put in the effort.

This could be as simple as you answering an ACM question for one of your peers while collaborating on a set or checking your sets with each other, and then asking your friend to ask that question on Piazza so you can share your answer with the class.
