My best recommendation for this class is to really make extra effort outside of the homework assignments to really learn the material. The lecture notes posted for this class are very thorough, so they are a very helpful resource for the homeworks. However, it can be easy to just do the assignments each week without fully internalizing the material. So, by the time the exams come around, that isn't the greatest. I personally found the exams to be quite difficult, but, I found that at least putting something down, even if it was just a thought process, would earn you partial credit, so make sure to do that if you ever encounter questions on the exam that you just don't know how to answer.

But yeah, PDEs are not super easy, so don't expect this class to be easy to pick up right away. If you ever are going to encounter PDEs in future classes/research, make sure to put time into this class outside of the homeworks to really get the most out of it.
ACM95b is kind of a troll class honestly. I had a lot of upperclassmen friends telling me that the class was pretty doable, especially with the help of Dan's slides. Overall, as someone who has struggled when the material in any Caltech math class starts to get too theoretical or expect you to derive things/accept crazy concepts and move on, there will be questions on the sets that are challenging. If you're not an ACM major, just make sure to keep up with the sets (getting extensions isn't hard and very useful to make sure you finish the set entirely!) so you're familiar with all the material you might see on a final and make sure to read all the lecture notes/slides, and a minimum B is probably very doable.
The first half of the course was fine in my opinion, and I found it much more enjoyable than 95A. However the second half was extremely hard to keep up with and the sets took quite a while to complete that extensions were needed on every set after midterms. The course content posted tended to help for the most part but be prepared to spend a lot of time on the sets in the latter half, and try to start them early and attend office hours if you can. Best of luck!
