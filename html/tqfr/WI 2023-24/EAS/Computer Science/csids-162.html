Comments

What would you tell a student thinking about taking this course in the future?

I actually enjoyed this class even though it was a bit different from what I initially expected. We created and iterated over a speculative algorithm/world/data we created to tell a story. We also discussed a lot of papers and articles about ethics which was my favorite part. I don't think there are many classes that talk about our responsibility to the world as scientists, so I highly recommend. We had the project we work on throughout the term, a final reflection, 2 readings every week, DQRS on those readings, and DQs on those readings as well that we post on ed stem discussion.
I thought that this course was not bad. It is pretty chill and the professors are nice, but sometimes the class can feel disorganized and unclear. The projects encourage you to think in different ways and it's not really like a traditional advanced CS, more like a VC course.
The course was alright but the vagueness of the project instructions and expectations made it a bit frustrating at times. Instructors were also pretty strict with attendance even though class time did not always feel valuable.
This was an interesting but somewhat strange advanced CS class. The premise of the class is a term-long project where we attempt to produce artistic content from data and mathematical models. First, note that this class has very little to do with "algorithms" in the CS sense. The "algorithmic" part of this class supposedly refers to the method through which we ideate and iterate our project. The amount of actual coding you do in this class is basically entirely up to you - I saw groups whose coding portion probably did not amount to more than writing a few matplotlib commands, while other groups probably had to write significantly more code for simulations, models, etc. The professors stress that the project does not turn into a "scientific" or "social science" one; rather, we should be treating this project as an exercise in "creativity" and utilize absurdity to underscore societal issues and build a cohesive narrative. I would say this class is actually rather front-loaded - the project is split up into several "steps" (checkpoints) which are due at various weeks over the quarter. The bulk of the work is done in Step 3, which is due by the end of Week 7. As a result, there isn't really that much left to do for the last few weeks. Class is more or less mandatory due to discussion and in-class activities, though it seemed like several people were absent on any given day. Overall, this was a very chill advanced CS class (likely easier than any other CS class which could be used to fulfill the 72 unit advanced CS requirement) and a cool space for exploring a subject you're interested in with computational tools and a more creative angle.
